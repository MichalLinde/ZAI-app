## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/lyes-mersel/countries-challenge.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Launch the server
   ```sh
   npm run dev
   ```
